# Meeting Cost Calculator

## Overview
This document tracks the instructions, steps, and fixes/changes for building the Meeting Cost Calculator solution.

A tool to calculate the true cost of meetings based on attendee salaries, duration, and other factors. Includes warnings for inefficient meetings and suggestions for async alternatives.

---

## Instructions

1. Set up project folder structure
2. Gather research data and requirements
3. Build the core calculator functionality
4. Implement warning triggers (social loafing, C-suite, cost thresholds)
5. Add output metrics (single meeting cost, annual projections)
6. Create user interface
7. Test and refine
