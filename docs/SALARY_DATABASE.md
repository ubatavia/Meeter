## Manual Override Template

When the default rates do not match a user's organization, the following fields are available for manual configuration:

| Field | Description | Default | Editable |
|-------|-------------|---------|----------|
| Custom Role Name | User-defined role label | — | Yes |
| Annual Salary (CAD) | Override salary for the role | From database | Yes |
| Hourly Rate (CAD) | Auto-calculated or manual entry | Annual / 2080 | Yes |
| Working Hours/Year | Adjust for part-time or contract | 2080 | Yes |
| Currency | Switch between CAD/USD/other | CAD | Yes |
| Benefits Multiplier | Include overhead costs (1.0 - 1.5x) | 1.0 | Yes |

## Notes

- All salaries in Canadian Dollars (CAD)
- Hourly rate = Annual Salary / 2,080 working hours
- Per minute = Hourly Rate / 60
- Per second = Hourly Rate / 3,600
- Salary data reflects national Canadian averages; regional variances apply (Toronto/Vancouver typically 10-20% higher)
- Benefits multiplier can be applied to reflect total compensation cost (typically 1.25x - 1.4x of base salary)
- Data should be refreshed annually to reflect market changes
